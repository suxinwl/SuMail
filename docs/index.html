⚡ 快速开始

三步完成部署

1. 下载并解压
# Linux (amd64)
wget https://github.com/.../goemail-linux-amd64.tar.gz
tar -xzf goemail-linux-amd64.tar.gz
2. 配置 (可选)
# 复制示例配置并修改
cp config.example.json config.json
vim config.json
3. 启动服务
./goemail

# 默认账号: admin / 123456
# 访问 http://localhost:8080
